## Query from S3
S3에 있는 데이터를 Athena로 데이터 쿼리

JSON 기반 데이터의 TABLE 생성 예제
```sql
CREATE EXTERNAL TABLE payments (
    id INT,
    user_id INT,
    name STRING,
    amount DOUBLE,
    status STRING,
    country STRING,
    event_time TIMESTAMP,
    date DATE
)
PARTITIONED BY (year int, month int, day int)
ROW FORMAT SERDE 'org.openx.data.jsonserde.JsonSerDe'
LOCATION 's3://demo-bucket-346151107821-ap-northeast-2-an/data'
```

파티션 추가
```sql
ALTER TABLE payments ADD PARTITION (
    year = 2025,
    month = 01,
    day = 01
)
LOCATION 's3://demo-bucket-346151107821-ap-northeast-2-an/data/year=2025/month=01/day=01/'
```

자동으로 파티션 추가
```sql
MSCK REPAIR TABLE payments 
```

projection으로 테이블 생성 시 자동 파티셔닝

```sql
CREATE EXTERNAL TABLE payments (
    id INT,
    user_id INT,
    name STRING,
    amount DOUBLE,
    status STRING,
    country STRING,
    event_time TIMESTAMP,
    date DATE
)
PARTITIONED BY (year int, month int, day int)
ROW FORMAT SERDE 'org.openx.data.jsonserde.JsonSerDe'
LOCATION 's3://demo-bucket-346151107821-ap-northeast-2-an/data'
TBLPROPERTIES (
    'projection.enabled' = 'true',
    'projection.year.type' = 'integer', 
    'projection.year.range' = '2025,2026',
    'projection.month.type' = 'integer',
    'projection.month.range' = '1,12',
    'projection.month.digits' = '2',
    'projection.day.type' = 'integer',
    'projection.day.range' = '1,31',
    'projection.day.digits' = '2',
    'storage.location.template' = 's3://demo-bucket-346151107821-ap-northeast-2-an/data/${year}/${month}/${day}/'
)
```

## 파일 공유 시스템

### 종류
- EFS(Elastic File System)
- S3FS

여러 서버, 인스턴스에서 동시에 접근 가능 (네트워크 연결 필수)

### S3-FS 설치

```bash
wget https://s3.amazonaws.com/mountpoint-s3-release/latest/x86_64/mount-s3.rpm
sudo dnf install ./mount-s3.rpm
```

```bash
mkdir data
mount-s3 <S3-name> data
```

#### 재시작 후에도 mount 유지하기

`/etc/fstab`

```
s3://amzn-s3-demo-bucket/example-prefix/ /mnt/mountpoint mount-s3 _netdev,nosuid,nodev,nofail,rw 0 0
```

- Role-A: efs 마운트 가능 읽기만 가능
- Role-B: efs 마운트 가능 (읽기/쓰기 가능)

2. ec2-user로 엑세스 가능해야 함
